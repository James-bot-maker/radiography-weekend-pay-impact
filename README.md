# Weekend Working Pay Impact (Streamlit)

Illustrative calculator to help staff compare the pay impact of:
- Current system: optional **bank Sundays**
- New system: **contracted Sundays** with **60% enhancement** (paid for 12 hours)

## Disclaimer
This tool is illustrative and uses simplified assumptions (tax/NI/pension/leave). It cannot represent individual circumstances.
Always check payslips, HR guidance, and your personal situation.

## Run locally
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
streamlit run app.py
